TRAINING VIDEO SCENARIO
=======================

This folder contains the materials for the Training Video scenario.

INCLUDED:
- roaster_calibration_script.txt - The script/transcript of Marcus's
  training video about calibrating the coffee roaster

TO USE THIS SCENARIO:

Option A: Use the script directly
- Read roaster_calibration_script.txt as if it were a transcript
- Skip the cc-transcribe step and go straight to creating documentation

Option B: Record your own video
- Record yourself (or anyone) reading the script
- Save as roaster_calibration.mp4
- Use cc-transcribe to extract the audio
- This tests the full workflow

Option C: Generate audio from text
- Use cc-voice or another text-to-speech tool
- Convert the script to audio
- Then use cc-transcribe to get it back to text
- Shows the full round-trip workflow

The key learning is: turning unstructured video/audio into useful
documentation. The source material doesn't matter - the process does.
