# Demo Files

This folder contains sample files for the tool showcase demos.

## Files to Create

### sample_video.mp4
Create a short (30-60 second) video for the cc-transcribe demo.

Options:
1. Record yourself saying something simple
2. Use a screen recording with voiceover
3. Download a royalty-free clip

The content should be clear speech that can be transcribed accurately.

### receipt_photo.jpg (for cc-ocr demo)
Take a photo of a receipt or printed document to demonstrate OCR.
