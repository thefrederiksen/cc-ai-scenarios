# Bookkeeping Task

Complete the 2025 bookkeeping for Mountain Pine Coffee Roasters.

All the receipts, invoices, and bank statements are in `01-bookkeeper/Mountain Pine Coffee Roasters/2025/`. Use `01-bookkeeper/Mountain Pine Coffee Roasters/2024/Bookkeeping 2024.xlsx` as a template for the format and categories.

1. Go through every document (PDFs and images) and extract the transaction data
2. Create a new `01-bookkeeper/Mountain Pine Coffee Roasters/2025/Bookkeeping 2025.xlsx` with the same structure as 2024
3. Enter all the transactions and make sure everything balances with the bank statements
4. Flag any issues you find (missing receipts, duplicates, amounts that don't match)

Save your work as you go.
