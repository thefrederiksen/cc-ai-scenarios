SCREENSHOT GUIDE SCENARIO
=========================

This folder is where you'll save screenshots as you navigate
the website setup process.

HOW TO USE THIS SCENARIO:

1. Open the website you need help with (e.g., Square, Shopify, Stripe)

2. Take a screenshot of the first confusing page
   - Windows: Win+Shift+S (Snipping Tool)
   - Mac: Cmd+Shift+4

3. Save the screenshot here as screenshot_01.png

4. In Claude Code, share the screenshot:
   "I need help with this page: [drag screenshot into chat]"

5. Claude will explain what each option means and what to select

6. After making changes, screenshot the next page as screenshot_02.png

7. Continue until setup is complete

SUGGESTED REAL PLATFORMS TO TRY:

1. Square Online (squareup.com/online-store)
   - Free tier available
   - Realistic e-commerce setup

2. Shopify (shopify.com)
   - 14-day free trial
   - Complex settings to navigate

3. Stripe Dashboard (dashboard.stripe.com)
   - Payment processing settings
   - Confusing for beginners

4. Google Workspace Admin (admin.google.com)
   - Email and domain settings
   - Many confusing options

The value of this scenario is learning that Claude can help you
navigate ANY confusing website, not just these specific ones.

PRIVACY NOTE:

Be careful not to include sensitive information in screenshots:
- API keys
- Passwords (use browser's "show password" feature carefully)
- Credit card numbers
- Social security numbers

You can blur or crop these before sharing if needed.
